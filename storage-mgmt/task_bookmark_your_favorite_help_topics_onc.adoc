= Setzen Sie ein Lesezeichen für Ihre bevorzugten Hilfethemen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Auf der Registerkarte „Hilfefavoriten“ können Sie häufig verwendete Hilfethemen mit einem Lesezeichen versehen.  Mithilfe von Lesezeichen können Sie schnell auf Ihre Lieblingsthemen zugreifen.

.Schritte
. Navigieren Sie zu dem Hilfethema, das Sie als Favorit hinzufügen möchten.
. Klicken Sie auf *Favoriten* und dann auf *Hinzufügen*.
