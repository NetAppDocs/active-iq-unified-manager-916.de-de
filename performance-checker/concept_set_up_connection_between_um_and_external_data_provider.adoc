= Einrichten einer Verbindung zwischen einem Unified Manager-Server und einem externen Datenanbieter
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Über eine Verbindung zwischen einem Unified Manager-Server und einem externen Datenanbieter können Sie Cluster-Leistungsdaten an einen externen Server senden, sodass Speichermanager die Leistungsmetriken mithilfe von Drittanbietersoftware grafisch darstellen können.

Eine Verbindung zwischen einem Unified Manager-Server und einem externen Datenanbieter wird über die Menüoption „Externer Datenanbieter“ in der Wartungskonsole hergestellt.
