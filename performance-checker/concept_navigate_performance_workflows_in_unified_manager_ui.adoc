= Navigieren Sie durch Leistungsworkflows in der Unified Manager-GUI
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Die Unified Manager-Schnittstelle bietet viele Seiten zum Sammeln und Anzeigen von Leistungsinformationen.  Sie verwenden das linke Navigationsfeld, um zu Seiten in der GUI zu navigieren, und Sie verwenden Registerkarten und Links auf den Seiten, um Informationen anzuzeigen und zu konfigurieren.

Sie verwenden alle folgenden Seiten, um Informationen zur Clusterleistung zu überwachen und Fehler zu beheben:

* Dashboard-Seite
* Seiten zum Speicher- und Netzwerkobjektinventar
* Detailseiten zu Speicherobjekten (einschließlich des Performance-Explorers)
* Konfigurations- und Setup-Seiten
* Veranstaltungsseiten
