= Wenden Sie Filter an, um die Liste der korrelierten Objekte im Raster zu reduzieren
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Durch Filtern können Sie eine kleinere, besser definierte Teilmenge von Objekten im Raster anzeigen.  Wenn Sie beispielsweise 25 Volumes im Raster haben, können Sie durch Filtern nur die Volumes anzeigen, deren Durchsatz unter 90 MBps liegt oder deren Latenz über 1 ms/Operation liegt.
