= Was sind Performance-Ereignisse
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Leistungsereignisse sind Vorfälle im Zusammenhang mit der Arbeitslastleistung auf einem Cluster.  Sie helfen Ihnen, Arbeitslasten mit langsamen Reaktionszeiten zu identifizieren.  Zusammen mit den gleichzeitig aufgetretenen Integritätsereignissen können Sie die Probleme ermitteln, die möglicherweise die langsamen Reaktionszeiten verursacht oder dazu beigetragen haben.

Wenn Unified Manager mehrere Vorkommen derselben Ereignisbedingung für dieselbe Clusterkomponente erkennt, behandelt er alle Vorkommen als ein einzelnes Ereignis und nicht als separate Ereignisse.
