= Festlegen eines Zeitbereichs für korrelierte Objekte
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Mit der Zeitbereichsauswahl auf der Seite „Performance Explorer“ können Sie den Zeitbereich für den Objektdatenvergleich angeben.  Durch die Angabe eines Zeitbereichs wird der Inhalt der Performance Explorer-Seiten verfeinert, sodass nur die Objektdaten innerhalb des von Ihnen angegebenen Zeitbereichs angezeigt werden.

Durch die Verfeinerung des Zeitbereichs können Sie auf effiziente Weise nur die Leistungsdaten anzeigen, die Sie interessieren.  Sie können einen vordefinierten Zeitraum auswählen oder einen benutzerdefinierten Zeitraum angeben.  Der Standardzeitraum sind die vorangegangenen 72 Stunden.
