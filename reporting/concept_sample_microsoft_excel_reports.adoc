= Beispiele für Microsoft Excel-Berichte
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Diese Beispielberichte zu Microsoft Excel sollen die Berichtsoptionen vorstellen, die mit den erweiterten Funktionen von Excel verfügbar sind.

Mit den erweiterten Funktionen von Excel können Sie eine große Bandbreite an Berichten erstellen, die speziell auf Ihre Anforderungen zugeschnitten sind.  Ausführliche Informationen zur Verwendung von Excel finden Sie in der Produktdokumentation.

[NOTE]
====
Zum Verwalten von Berichten müssen Sie über die Rolle „Anwendungsadministrator“ oder „Speicheradministrator“ verfügen.

====
