= Anpassen von NFS-Freigabeberichten
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können NFS-Freigabeberichte anpassen, um Informationen zu NFS-Exportrichtlinien und -Regeln für Volumes auf Ihren Speichersystemen zu analysieren.  Sie können beispielsweise Berichte anpassen, um Volumes mit nicht zugänglichen Bereitstellungspfaden und Volumes mit der Standardexportrichtlinie anzuzeigen.
