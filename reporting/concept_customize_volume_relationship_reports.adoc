= Anpassen von Volumenbeziehungsberichten
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Mit dem Bericht „Volume Relationships Inventory“ können Sie die Speicherinventardetails in einem Cluster analysieren, den für Volumes erforderlichen Schutzgrad ermitteln und die Volumedetails basierend auf Fehlerquelle, Muster und Zeitplänen filtern.
