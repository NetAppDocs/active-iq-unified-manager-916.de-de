= Upgrade von Produkten von Drittanbietern
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können Produkte von Drittanbietern, wie etwa JRE, auf Unified Manager aktualisieren, wenn diese auf Windows-Systemen installiert sind.

Die Unternehmen, die diese Produkte von Drittanbietern entwickeln, melden regelmäßig Sicherheitslücken.  Sie können nach Ihrem eigenen Zeitplan auf neuere Versionen dieser Software aktualisieren.
