= Seite „Skripte“
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Auf der Seite „Skripte“ können Sie Ihre benutzerdefinierten Skripte zu Unified Manager hinzufügen.  Sie können diese Skripte mit Warnungen verknüpfen, um die automatische Neukonfiguration von Speicherobjekten zu ermöglichen.

Auf der Seite „Skripte“ können Sie Skripte zum Unified Manager hinzufügen oder daraus löschen.



== Befehlsschaltflächen

* *Hinzufügen*
+
Zeigt das Dialogfeld „Skript hinzufügen“ an, in dem Sie Skripte hinzufügen können.

* *Löschen*
+
Löscht das ausgewählte Skript.





== Listenansicht

Die Listenansicht zeigt die Skripte, die Sie zu Unified Manager hinzugefügt haben, in tabellarischer Form an.

* *Name*
+
Zeigt den Namen des Skripts an.

* *Beschreibung*
+
Zeigt die Beschreibung des Skripts an.
