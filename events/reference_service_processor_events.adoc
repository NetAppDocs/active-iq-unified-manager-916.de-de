= Serviceprozessorereignisse
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Serviceprozessorereignisse liefern Ihnen Informationen zum Status Ihres Prozessors, sodass Sie ihn auf potenzielle Probleme überwachen können.  Ereignisse werden nach Auswirkungsbereich gruppiert und enthalten den Ereignis- und Trap-Namen, die Auswirkungsstufe, den Quellentyp und den Schweregrad.

 a|

 a|

|===
