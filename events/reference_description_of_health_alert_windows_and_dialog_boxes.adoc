= Beschreibung der Warnfenster und Dialogfelder
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie sollten Warnungen konfigurieren, um Benachrichtigungen über Ereignisse zu erhalten, indem Sie das Dialogfeld „Warnung hinzufügen“ verwenden.  Sie können die Liste der Warnungen auch auf der Seite „Warnungseinrichtung“ anzeigen.
