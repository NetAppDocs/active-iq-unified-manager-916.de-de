= Aktualisieren Sie Produkte von Drittanbietern nach der Installation von Unified Manager
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können Produkte von Drittanbietern wie JRE aktualisieren, wenn Unified Manager bereits auf Linux-Systemen installiert ist.

Die Unternehmen, die diese Produkte von Drittanbietern entwickeln, melden regelmäßig Sicherheitslücken.  Sie können nach Ihrem eigenen Zeitplan auf neuere Versionen dieser Software aktualisieren.
