= Entfernen Sie den benutzerdefinierten Umadmin-Benutzer und die Wartungsgruppe
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Wenn Sie vor der Installation von Unified Manager ein benutzerdefiniertes Home-Verzeichnis erstellt haben, um Ihren eigenen Umadmin-Benutzer und Ihr Wartungskonto zu definieren, sollten Sie diese Elemente nach der Deinstallation von Unified Manager entfernen.

Bei der Standarddeinstallation von Unified Manager werden benutzerdefinierte Umadmin-Benutzer und Wartungskonten nicht entfernt.  Sie müssen diese Elemente manuell löschen.

.Schritte
. Melden Sie sich als Root-Benutzer beim Red Hat Enterprise Linux-Server an.
. Löschen Sie den umadmin-Benutzer:
+
`userdel umadmin`

. Löschen Sie die Wartungsgruppe:
+
`groupdel maintenance`
