= Jobdetails anzeigen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Nachdem Sie einen Auftrag gestartet haben, können Sie seinen Fortschritt auf der Seite „Auftragsdetails“ verfolgen und die zugehörigen Aufgaben auf mögliche Fehler überwachen.

.Bevor Sie beginnen
Sie müssen über die Rolle „Anwendungsadministrator“ oder „Speicheradministrator“ verfügen.

.Schritte
. Klicken Sie im linken Navigationsbereich auf *Schutz* > *Jobs*.
. Klicken Sie auf der Seite „Jobs“ in der Spalte „Name“ auf einen Jobnamen, um die Liste der mit dem Job verknüpften Aufgaben anzuzeigen.
. Klicken Sie auf eine Aufgabe, um zusätzliche Informationen im Bereich *Aufgabendetails* und im Bereich *Aufgabennachrichten* rechts neben der Aufgabenliste anzuzeigen.
