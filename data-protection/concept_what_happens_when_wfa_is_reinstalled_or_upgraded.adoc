= Was passiert, wenn OnCommand Workflow Automation neu installiert oder aktualisiert wird?
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Bevor Sie OnCommand Workflow Automation neu installieren oder aktualisieren, müssen Sie zunächst die Verbindung zwischen OnCommand Workflow Automation und Unified Manager entfernen und sicherstellen, dass alle derzeit ausgeführten oder geplanten Jobs von OnCommand Workflow Automation gestoppt werden.

Sie müssen Unified Manager auch manuell aus OnCommand Workflow Automation löschen.

Nachdem Sie OnCommand Workflow Automation neu installiert oder aktualisiert haben, müssen Sie die Verbindung mit Unified Manager erneut einrichten.
