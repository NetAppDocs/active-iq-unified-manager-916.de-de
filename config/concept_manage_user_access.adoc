= Verwalten des Benutzerzugriffs
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können Rollen erstellen und Berechtigungen zuweisen, um den Benutzerzugriff auf Active IQ Unified Manager zu steuern.  Sie können Benutzer identifizieren, die über die erforderlichen Berechtigungen verfügen, um auf ausgewählte Objekte in Unified Manager zuzugreifen.  Nur Benutzer mit diesen Rollen und Berechtigungen können die Objekte in Unified Manager verwalten.
